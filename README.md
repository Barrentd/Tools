# Tools
I create a makefile to install my environement with apt.

### Dependency required :

- git (to clone the project)
- make (to execute the makefile)
- sudo (to grant the user root privilege)

### The installation file includes :

- gnome-shell
- gnome-terminal
- vim
- nano
- htop
- curl
- nautilus
- firefox
- keepassx
- terminator
